# AGENTS.md

해당 디렉토리는 AWS, K8S 등 인프라 자원을 프로비저닝하기 위한 terraform 코드 프로젝트들의 최상위 디렉토리이다.
해당 디렉토리 하위의 각 목적별 terraform 프로젝트에서 작업할 때는 반드시 아래 규칙을 따라야 한다.

## 기본 규칙
- 모든 `terraform.tfvars`, `*.pem` 파일과 `ssh/`, `.terraform/` 디렉토리는 읽거나 수정하지 않는다.

## terraform 규칙
- state는 직접 수정하지 않는다. 단, 예외적으로 필요한 경우 확인을 받고 수행할 수 있다.
- terraform provider가 필요한 경우에는 provider 이름과 이유를 설명하고 확인을 기다린다.
- 명시적인 요청/허락이 없는 한 리소스를 삭제하지 않는다.
- 재사용 가능한 코드는 모듈로 빼는 것을 선호한다.
- 새로운 요소가 추가/삭제될 때 resource key가 변경될 수 있는 문제를 방지하기 위해, 리소스 동적 생성 시 list 대신 map을 우선해서 사용한다.
